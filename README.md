# 📖 About

**Delicious Foods** is a web-based food ordering application that allows customers to browse food items, place orders, and manage their accounts through a simple and user-friendly interface.

The project is developed using **Java, JSP, MySQL, HTML, CSS, and JavaScript**.

---

# ✨ Features

✅ User Registration & Login

✅ Browse Food Menu

✅ Add Items to Cart

✅ Place Orders

✅ View Order History

✅ Admin Dashboard

✅ Database Integration
