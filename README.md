# 📖 Overview

**Delicious Foods** is a web-based food ordering platform designed to provide customers with a simple and efficient way to browse menus, place food orders, and manage their accounts.

The application includes both customer and administrator modules, offering complete food ordering and management functionality.

---

# ✨ Features

- 👤 User Registration & Login
- 🍔 Browse Food Menu
- 🔍 Search Food Items
- 🛒 Add Items to Cart
- 📦 Place Orders
- 📜 Order History
- ❤️ User-Friendly Interface
- 🔐 Secure Authentication
- ⚙️ Admin Dashboard
- 🗄️ MySQL Database Integration

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/karankumar-dsu/delicious-foods.git
```

### Setup

1. Import the project into Eclipse or IntelliJ IDEA.
2. Configure Apache Tomcat Server.
3. Import the MySQL database from the `database` folder.
4. Update database credentials.
5. Run the project on Tomcat.
